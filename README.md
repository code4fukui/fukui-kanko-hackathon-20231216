# fukui-kanko-hackathon-20231216

## Overview
A hackathon "Create Next-Generation Tourism with Civic Tech" hosted by the Fukui Tourism DX Promotion Marketing Data Consortium and Code for JAPAN.

## Features
- Develop innovative tourism apps for Fukui prefecture using open data
- Discover and nurture digital talent for tourism DX in Fukui
- Create new DX initiatives to revitalize tourism in Fukui

## Participation
- Target participants: middle/high school students, college students, general public
- Date: December 16, 2023 (Sat), 1:00 PM - 6:00 PM
- Venue: jig.jp Development Center, Sabae City

## Prizes
- Participation prizes
- Grand prize: Meta Quest 3 VR headset for the winning team
