# fukui-kanko-hackathon-20231216

福井の観光DXを推進するためのハッカソンのリポジトリです。

## デモ
イベントの詳細は[こちらのページ](https://code4fukui.github.io/fukui-kanko-hackathon-20231216/)で確認できます。

## 概要
- 福井県の観光DX人材の発掘と育成
- 新しい福井県の観光DXソリューションの創造

## 参加対象
- 中学生、高校生、高専生、一般

## 開催日時
- 2023年12月16日(土) 13:00-18:00

## 会場
- jig.jp開発センター(鯖江市)
